# Manuals

`manual_src.html` (Japanese) and `manual_src_en.html` (English) are the sources of the user manuals.
They reference `screenshots/*.png`, which are **not** committed.

The manuals must be rebuilt from screenshots taken on a **public, redistributable CT dataset**
(HCC-TACE-Seg, The Cancer Imaging Archive, CC BY 4.0 — see the repository README).
No patient imaging from any institution may be committed to this repository or embedded in the PDFs.

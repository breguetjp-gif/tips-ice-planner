# TIPS ICE Planner（Windows版）— はじめにお読みください

術前CTから **ICE（心腔内エコー）の見え方** と **穿刺ライン** を“素振り”する、TIPS術前プランニング・ツールです。

> ⚠️ **研究・教育・自己研鑽用です。医療機器ではありません。術中ナビゲーションではありません。最終的な臨床判断は必ず術者が行ってください。**
> 「推奨穿刺経路」「最適穿刺点」「穿刺回数の削減」を示すものではありません。

## 動作環境
- Windows 10 / 11（64-bit）。**Python等の事前インストールは不要**（自己完結アプリ）。

## 導入
1. zip を解凍。
2. **「TIPS ICE Planner」フォルダを丸ごと**、任意の場所（例: デスクトップ）に置く。
   - ※ `TIPS ICE Planner.exe` は同梱の `_internal` フォルダが必要です。**exe単体を取り出さず、フォルダごと**扱ってください。
3. フォルダ内の **`TIPS ICE Planner.exe` をダブルクリック**。
4. 初回に「**Windows によって PC が保護されました（SmartScreen）**」が出たら、
   **「詳細情報」→「実行」** を押してください（現在は無署名のため。院内配布での運用です）。

## 使い方（概要）
1. **Database**（入口）：`Import DICOM folder…` で症例取り込み → 一覧＋サムネイル → ダブルクリックで開く。コメント欄に教育用メモを残せます。
2. **Step 1（ICE setup）**：Axial をクリックして IVC パス → 挿入方向（Femoral=足 / Jugular=頸）→ θ／Probe／Deflect で ICE 像を見回す。
3. **Step 2（Needle）**：Entry → Target を置き、Needle curve で弓なり調整。針が Axial/Coronal/Sagittal/ICE に表示。点はドラッグ移動可。
- 拡大＝右ドラッグ、移動＝中ドラッグ、リセット＝右クリック。

## プライバシー
- 患者DICOMは**この端末内のみ**で処理され、外部に送信されません。

# TIPS Planner — ICE 穿刺プランナー（Miele-LXIV プラグイン）

術前の造影 CT から、TIPS（経頸静脈的肝内門脈大循環短絡術）の **ICE（心腔内エコー）の見え方** と **穿刺ライン** を“素振り”できる、Miele-LXIV 用の研究・教育プラグインです。

**対象**: Miele-LXIV（Mac App Store 版）/ macOS（Apple Silicon・Intel 両対応）
**作成**: M. Yamamoto（個人開発）
**ライセンス**: GPL-3.0（土台の Miele-LXIV に準拠）

## これは何か

患者の造影 CT（DICOM）を Miele-LXIV で開き、

1. **Step 1 — ICE セットアップ**: Axial をクリックして IVC（ICE プローブ）の経路を引き、**挿入方向（足から＝Femoral / 頸から＝Jugular）** を選んで先端（TIP）を確定。
2. **Step 2 — 針プラン**: 刺入点（Entry）と標的（Target）を置き、針の種類と曲げ（Curve）を選ぶと、**Axial / Coronal / Sagittal / ICE / 3D の各ビューに針経路を表示**。

ICE の扇は先端アレイから出る側射断面として、θ 回転・偏向・前後送りに連動して動きます。**3D linkage** ビューでプローブ・扇・針の立体的な位置関係を自由回転で確認できます。

## 導入手順

### 前提
- **Miele-LXIV（Mac App Store 版）** を導入済みであること。

### インストール
1. **Miele-LXIV を一度起動して終了**（初回フォルダ作成のため）。
2. Finder で **Cmd + Shift + G**、次のパスを開く：
   ```
   ~/Library/Containers/com.bettarini.miele-lxiv/Data/Library/Application Support/miele-lxiv/Plugins/
   ```
3. 配布物の **`TIPSPlanner.mieleplugin`** をこのフォルダにドロップ。
4. **Miele-LXIV を再起動**。
5. CT を 1 つ開き、メニュー **Plugins → TIPS Planner**。初回は研究・教育用の注意ダイアログが出ます。

> 署名・公証前の配布で「壊れている」と出る場合のみ、ターミナルで一度：
> ```
> xattr -dr com.apple.quarantine "~/Library/Containers/com.bettarini.miele-lxiv/Data/Library/Application Support/miele-lxiv/Plugins/TIPSPlanner.mieleplugin"
> ```

## 使い方

### Step 1: ICE path（プローブ経路）
- **Axial をクリック**して IVC に沿って数点（2 点以上）。水色の線で経路が表示されます。
- **挿入方向**を選択：`Femoral（足から）` / `Jugular（頸から）`。これで **先端（TIP・青）** が確定し、屈曲の向きが正しくなります。
- ここで止めて ICE 像だけ見る使い方も可。

### Step 2: Needle path（針プラン）
- `Entry` を選んで刺入点 → 自動で `Target` → 標的をクリック。**緑＝Entry / 赤＝Target**。
- 針の操作は `Needle curve`（弓なり角）のみ（針種の選択は廃止）：`0°`＝直線、レンジ `−20°`〜`+30°`（既定 `+20°`）。正負で弓なりの向きが変わります。数値表示。
- **針は常に Entry↔Target を結び、4 面＋ICE に点線で表示**。ICE には「×＝針が断面を貫く位置」。
- **Entry / Target の丸は、どのビューでも掴んでドラッグで移動**できます（CT の経路を見ながら調整）。

### ビューの操作
| 操作 | 効果 |
|---|---|
| 左ドラッグ | 階調（横=WW・縦=WL） |
| 右ドラッグ / ピンチ / ホイール | 拡大縮小（カーソル中心）※ICE はホイール=θ回転 |
| 二本指スクロール | 断面送り |
| 右クリック | 拡大・移動をリセット |
| 境界ドラッグ | 各ペインのサイズ変更 |
| θ / Probe / Deflect A-P・L-R | 回転・前後送り・先端偏向 |
| Rotate 90° / Zero deflection / Flip L/R (ICE) | 表示回転 / 偏向リセット / ICE 左右反転 |
| 3D linkage | ドラッグで自由回転（灰=シャフト / 橙=偏向で曲がる先端 / 青=アレイ / シアン=扇（アレイ中央から） / 橙点線=針） |

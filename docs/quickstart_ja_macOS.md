# TIPS ICE Planner（Mac版）— はじめにお読みください

術前CTから **ICE（心腔内エコー）の見え方** と **穿刺ライン** を“素振り”する、TIPS術前プランニング・ツールです。

> ⚠️ **研究・教育・自己研鑽用です。医療機器ではありません。術中ナビゲーションではありません。最終的な臨床判断は必ず術者が行ってください。**
> 「推奨穿刺経路」「最適穿刺点」「穿刺回数の削減」を示すものではありません。

## 動作環境
- macOS 11 以降（Apple Silicon / Intel）。**Python等の事前インストールは不要**（自己完結アプリ）。

## 導入
1. zip を解凍。
2. **「TIPS ICE Planner」を「アプリケーション」フォルダにドラッグ**。
3. ダブルクリックで起動（**Developer ID署名＋Apple公証済みなので警告は出ません**）。

## 使い方（概要）
1. **Database**（入口）：`Import DICOM folder…` で症例を取り込み → 一覧＋サムネイル → シリーズをダブルクリックで開く。コメント欄に教育用メモを残せます。
2. **Step 1（ICE setup）**：Axial をクリックして IVC パスを引く → 挿入方向（Femoral=足 / Jugular=頸）を選ぶ → θ／Probe／Deflect で ICE 像を見回す。
3. **Step 2（Needle）**：Entry → Target を置き、Needle curve で弓なりを調整。針が Axial/Coronal/Sagittal/ICE に表示されます。Entry/Target は各画面でドラッグ移動可。
- 拡大＝右ドラッグ、移動＝中ドラッグ、リセット＝右クリック。

## プライバシー
- 患者DICOMは**この端末内のみ**で処理され、外部に送信されません。取り込んだ目録・サムネイルも端末内のローカルフォルダに保存されます。
